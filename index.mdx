RAG-Anything is an open-source Python library that brings multimodal intelligence to retrieval-augmented generation. Unlike text-only RAG systems, it handles the full spectrum of document content—diagrams, tables, mathematical equations, and rich text—through a single coherent pipeline. Built on [LightRAG](https://github.com/HKUDS/LightRAG), it combines hybrid vector-graph retrieval with specialized modal processors to deliver contextually accurate answers from complex, mixed-content documents.

## What RAG-Anything does

Modern technical documents — research papers, financial reports, engineering specs — mix text, charts, tables, and equations. Traditional RAG systems drop most of that richness. RAG-Anything preserves it by routing each content type through a dedicated processor, then building a multimodal knowledge graph that captures relationships across modalities.

## Get started

<Steps>
  <Step title="Install RAG-Anything">
    ```bash
    pip install raganything
    ```
    For extended format support (images, text files, PaddleOCR):
    ```bash
    pip install 'raganything[all]'
    ```
  </Step>
  <Step title="Configure your LLM and embedding functions">
    RAG-Anything wraps any OpenAI-compatible LLM and embedding endpoint. Set your API key and define the model functions — see the [Quickstart](/quickstart) for a complete example.
  </Step>
  <Step title="Process a document">
    ```python
    await rag.process_document_complete(
        file_path="research_paper.pdf",
        output_dir="./output"
    )
    ```
  </Step>
  <Step title="Query your knowledge base">
    ```python
    result = await rag.aquery(
        "What are the key findings in the figures and tables?",
        mode="hybrid"
    )
    ```
  </Step>
</Steps>
